### Order Entry AI Accuracy

The percentage of all processed line items that required no human edits.

### Total Lines Processed

The total number of line items processed by Endeavor during the selected time period.

### Lines by AI

The number of line items fully processed by Endeavor without any CSR intervention.

### Lines by Human

The number of line items where a CSR made any manual edit—indicating the model required correction.

### Extraction Accuracy

Percentage of lines where Endeavor successfully extracted all required fields from the document.

(Measures extraction → not product matching.)

### Matching Accuracy

Percentage of extracted lines that Endeavor correctly matched to the catalog without human edits.

Lines corrected by CSRs count as mismatches.

### Orders Processing

The number of orders currently in the middle of extraction and matching.

### Orders Queued

Orders waiting in the queue to begin processing.

### AI Accuracy Over Time

A time-series view showing how extraction and matching accuracy trends over the selected date range.

### AI Processing Time by File Type

Breakdown of average processing times for each file format (PDF, Excel, images, etc.).

Helps diagnose slow or unusually large file types.

### Average Round Trip Time

The average time from when an order is uploaded to when extraction + matching completes.
