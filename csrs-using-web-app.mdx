### Step 2: Login Credentials in Email

You should have received an email titled "Welcome to Omelas by Endeavor AI". There you will find your username and temporary password.

<img
  style={{ borderRadius: '0.5rem', width: '100%', opacity: 1 }}

### Step 5: Editing an Order

**To manually add or remove rows**

<img
  style={{ borderRadius: '0.5rem', width: '100%', opacity: 1 }}
  src="/images/edit.png"
  alt="Order items table with add/remove row controls"
/>

1. Use the "Add Item" to add a line. It will default to being added to the bottom of list.
2. Use the + or - toggle by *hovering* over a row.

**Highlighted Rows**

<img
  style={{ borderRadius: '0.5rem', width: '100%', opacity: 1 }}
  src="/images/h.png"
  alt="Order items table showing highlighted rows"
/>

- Yellow = Zero or negative quantity
- Blue = Edited lines
- Red = No match

**Header Information**

<img
  style={{ borderRadius: '0.5rem', width: '100%', opacity: 1 }}
  src="/images/headr.png"
  alt="Order Information section with form fields"
/>

Click the "Order Information" section and a dropdown will appear. Most of the information is extracted directly from the uploaded order.

- Company Name = Company Name listed on the order
- Customer Reference = associated customer ID from ERP
- Delivery Address = connected address to customer ID in ERP
- Requested Delivery Date = delivery date extracted from order
- Purchase Order Number = po number from order

### Step 6: Finalize Order & Send to ERP

Once all information is filled and you're ready to enter it into the ERP, click "Approve"

<img
  style={{ borderRadius: '0.5rem', width: '100%', opacity: 1 }}
  src="/images/approve.png"
  alt="Order approval interface with Enter into ERP button"
/>

The screen will turn to Read-only (which means you cannot edit).

- If you're ready to finalize the order to your ERP, click "Enter into ERP"

- If you realize a change needs to be made, click "Restore" on the bottom right

<img
  style={{ borderRadius: '0.5rem', width: '100%', opacity: 1 }}
  src="/images/enter-erp.png"
  alt="Read-only order view with Enter into ERP and Restore buttons"
/>
